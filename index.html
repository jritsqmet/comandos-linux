Linux Hardware Monitor

Comandos esenciales para diagnosticar y monitorear tu sistema Linux

  • Sistema
  • Disco
  • RAM
  • CPU
  • Temperatura
  • Gráficos
  • Monitoreo

Información del Sistema

Comando Descripción
uname -a Muestra información del sistema: kernel, arquitectura, nombre del host, etc.
lscpu Detalles de la CPU: núcleos, modelo, velocidad, arquitectura.
lshw Lista detallada de todo el hardware (requiere sudo).
lshw -short Versión resumida de lshw.
lspci Lista dispositivos PCI (gráficos, red, sonido, etc.).
lsusb Lista dispositivos USB conectados.
dmidecode Información del BIOS: RAM, placa base, etc. (sudo).
hostnamectl Información del hostname, SO, arquitectura.

Uso del Disco

Comando Descripción
df -h Uso de espacio en disco (formato legible: GB/MB).
du -sh /ruta/ Tamaño de un directorio (ej: du -sh /home).
du -ah /ruta/ | sort -rh | head -10 Los 10 archivos/carpetas más grandes en una ruta.
lsblk Muestra todas las particiones y dispositivos de bloque.
fdisk -l Lista particiones de disco (sudo).

Memoria RAM

Comando Descripción
free -h Uso de RAM y swap (formato legible).
cat /proc/meminfo Detalles técnicos sobre la memoria del sistema.
htop o top Procesos en tiempo real: CPU, RAM, etc. (htop más amigable).
vmstat Estadísticas de memoria, procesos, I/O, CPU.

CPU

Comando Descripción
lscpu Información detallada de la CPU.
cat /proc/cpuinfo Detalles técnicos de cada núcleo.
mpstat Estadísticas de uso de CPU (requiere sysstat).
uptime Tiempo de encendido y carga promedio del sistema.

Temperatura

Comando Descripción
sensors Muestra temperatura de CPU, GPU, disco (requiere lm-sensors).
sensors-detect Configura sensores de hardware (primera vez).
nvidia-smi Temperatura y uso de GPU NVIDIA.
watch -n 2 sensors Actualiza cada 2 segundos la lectura de temperatura.

Gráficos y GPU

Comando Descripción
lspci | grep -i vga Muestra la tarjeta gráfica.
glxinfo | grep "OpenGL renderer" Renderizador gráfico actual.
nvidia-smi Uso, temperatura y memoria de GPU NVIDIA.
vainfo Verifica aceleración por hardware (video).

Monitoreo en Tiempo Real

Comando Descripción
htop Monitor avanzado de procesos (CPU, RAM, carga).
glances Herramienta avanzada de monitoreo (instalable con pip).
nmon Monitor interactivo de rendimiento del sistema.
iotop Uso de disco por procesos (como top para I/O). 
# Ejemplo de instalación:
sudo apt install htop glances nmon iotop lm-sensors
